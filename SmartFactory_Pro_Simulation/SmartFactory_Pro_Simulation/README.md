# SmartFactory Pro

SmartFactory Pro is a frontend-based production tracking and factory management simulation for a steel factory environment.

## Project Purpose

The purpose of this project is to simulate a professional factory production monitoring system.  
It demonstrates how a software system can be used to track production records, machine performance, maintenance requests, user roles, reports, and dashboard analytics.

## Main Features

- Login screen simulation
- Admin dashboard
- Production tracking table
- Machine performance cards
- Maintenance request management
- KPI cards
- Weekly production chart
- Factory map simulation
- Live search
- Multi-filtering
- CSV report export
- Print / Save as PDF support
- User management section
- Recent activity log
- Dark / Light theme
- Responsive design

1. Download or clone the project.
2. Open the folder.
3. Double-click `index.html`.
4. Login with the demo credentials.
5. Use the dashboard and simulation screens.
