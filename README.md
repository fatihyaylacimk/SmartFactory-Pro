# SmartFactory Pro

SmartFactory Pro is a frontend-based production tracking and factory management simulation designed for a steel factory environment.

The project demonstrates how a modern software system can be used to monitor production records, machine performance, maintenance requests, KPI metrics, reports, and user activity from a professional dashboard interface.

---

## Project Purpose

The main purpose of this project is to simulate a real-world factory production tracking system.

In industrial environments, production data, machine status, maintenance records, and performance reports are important for decision-making. SmartFactory Pro provides a simple but professional simulation of how these processes can be managed through a web-based dashboard.

---

## Main Features

- Login screen simulation
- Admin dashboard
- Production tracking table
- Machine performance monitoring
- Maintenance request management
- KPI cards
- Weekly production chart
- Factory map simulation
- Live search
- Production line filtering
- Status filtering
- CSV report export
- Print / Save as PDF support
- User management section
- Recent activity log
- Dark / Light theme
- Responsive design for desktop and mobile devices
